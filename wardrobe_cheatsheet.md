# Wardrobe Cheat Sheet

**Style Target:** Clean, confident, effortless (casual + professional) without "trying too hard."

**Base colors:** Black / white / grey / navy  
**Optional accents:** Olive / tan / burgundy  
**Fit rules:** Shoulders align, sleeves end at wrist bone, pants not pooling

## Outfit Formulas

### Casual (Clean)
* T-shirt + dark jeans + clean sneakers
* Polo/OCBD + chinos + sneakers/loafers
* Crewneck + chinos + sneakers

### Professional (Effortless)
* OCBD + chinos + loafers/derby
* OCBD + blazer + chinos/trousers + smart shoes
* Dress shirt + trousers + blazer (events)

**Rule:** If you wear a blazer, keep everything else simple.

---

## Buying Checklist

Before buying:
* [ ] It fits your palette (neutral)
* [ ] It matches at least 3 items you already own
* [ ] It's not a "statement" unless you already have basics
* [ ] Fabric feels decent (not see-through, not itchy)
* [ ] You can wear it both casually and slightly smart (bonus)

### Priority Buy Order
1. [ ] Clean sneakers
2. [ ] Dark jeans + chinos
3. [ ] Plain tees
4. [ ] OCBD shirts
5. [ ] Simple jacket
6. [ ] Smart shoes + belt
7. [ ] Blazer (if needed)

---

## "Not Trying Too Hard" Rules

* Avoid loud logos
* Fit and cleanliness > trendy pieces
* Neutral base always works
* One nice layer (jacket/blazer) elevates instantly
* Keep accessories minimal
