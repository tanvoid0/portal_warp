## Complete Checklist Cheat Sheet (30yo man)

Built to look **clean, confident, effortless** (casual + professional) without “trying too hard.” Use as a checklist you can slowly complete.

---

# 0) The Weekly Rhythm (simple)

* **Daily (10–20 min total):** hygiene + one small upgrade + 10-min reset
* **Weekly (60–120 min):** laundry + wardrobe check + groceries + one workout “anchor”
* **Monthly (2–3 hours):** haircut + wardrobe audit + replace worn basics

---

# 1) Daily Checklist (copy this)

### Morning (7–12 min)

* [ ] Brush teeth + tongue
* [ ] Deodorant
* [ ] Face: rinse/cleanse + moisturize + SPF
* [ ] Hair: quick style (1 product)
* [ ] Clean clothes (underwear/socks guaranteed)
* [ ] Shoes check (clean-ish)

* [ ] Brush teeth + (floss if you can)

---

# 2) Wardrobe: “Effortless Professional” System

## A) Your style target (so you don’t overdo it)

**Simple, fitted, neutral, repeatable.**

* Base colors: **black / white / grey / navy**
* Optional accents: **olive / tan / burgundy**
* Fit rules: shoulders align, sleeves end at wrist bone, pants not pooling

---

## B) Starter Wardrobe (capsule that works)

You don’t need all at once. This is the “end state” to build toward.

### Casual essentials

* [ ] 3–5 plain t-shirts (white/black/grey/navy)
* [ ] 2–3 polos or casual shirts (oxford button-down works best)
* [ ] 1–2 jeans (dark blue, black)
* [ ] 1–2 chinos (navy, tan/stone)
* [ ] 1 hoodie/crewneck (neutral)
* [ ] 1 casual jacket (bomber/denim/field jacket—neutral)
* [ ] Sneakers: 1 clean pair (white or black)
* [ ] Casual belt (simple)

### Professional essentials (office, smart casual, events)

* [ ] 2 oxford shirts (white, light blue)
* [ ] 1–2 dress shirts (if you need them)
* [ ] 1 blazer OR structured jacket (navy/charcoal)
* [ ] 1 pair dress trousers (charcoal/navy)
* [ ] 1 pair smart shoes (loafers/derby—black or dark brown)
* [ ] 1 smart belt matching shoes
* [ ] 1 coat (wool coat or simple trench for cold seasons)

### Underwear / socks (unsexy but huge confidence)

* [ ] 10–14 underwear (same brand/style)
* [ ] 10–14 socks (same brand/style: black/white)
* [ ] 2–4 “nice” socks (subtle patterns for office)

### Optional “one statement”

* [ ] One watch OR one simple chain OR one ring (keep it minimal)

---

## C) Outfit formulas (never think again)

### Casual (clean)

* T-shirt + dark jeans + clean sneakers
* Polo/OCBD + chinos + sneakers/loafers
* Crewneck + chinos + sneakers

### Professional (effortless)

* OCBD + chinos + loafers/derby
* OCBD + blazer + chinos/trousers + smart shoes
* Dress shirt + trousers + blazer (events)

**Rule:** If you wear a blazer, keep everything else simple.

# 3) Wardrobe Organization Checklist (drawer + closet)

## Step 1 — Declutter (60–90 min once)

Make 4 piles:

* [ ] Keep (fits, worn in last year)
* [ ] Maybe (fits but unsure)
* [ ] Donate/Sell
* [ ] Trash (stained, ripped, stretched)

Quick filters:

* [ ] Does it fit now?
* [ ] Would I wear it this week?
* [ ] Would I buy it again today?

## Step 2 — Sort by category (30 min)

* [ ] T-shirts
* [ ] Shirts (button-downs)
* [ ] Pants (jeans/chinos/trousers)
* [ ] Layers (hoodies/jackets)
* [ ] Gym
* [ ] Sleepwear
* [ ] Accessories (belts, watch, etc.)

## Step 3 — Closet layout (the pro layout)

From left to right:

* [ ] Work shirts (white/blue)
* [ ] Casual shirts
* [ ] Blazer / coat
* [ ] Jackets
  Bottom:
* [ ] Shoes lined up (casual → smart)
* [ ] Basket for “worn but not dirty”

## Step 4 — Drawer rules (so it stays tidy)

* [ ] Underwear in one box
* [ ] Socks paired + stacked
* [ ] Tees file-folded vertically
* [ ] One “gym drawer” only
* [ ] Nothing random allowed (no cables, papers)

## Step 5 — Make 10 saved outfits (30 min)

* [ ] Create 5 casual outfits
* [ ] Create 5 work/pro outfits
* [ ] Save as Notes with a photo if possible

# 4) Buying Checklist (so you don’t waste money)

Before buying:

* [ ] It fits your palette (neutral)
* [ ] It matches at least 3 items you already own
* [ ] It’s not a “statement” unless you already have basics
* [ ] Fabric feels decent (not see-through, not itchy)
* [ ] You can wear it both casually and slightly smart (bonus)

Priority buy order:

1. [ ] Clean sneakers
2. [ ] Dark jeans + chinos
3. [ ] Plain tees
4. [ ] OCBD shirts
5. [ ] Simple jacket
6. [ ] Smart shoes + belt
7. [ ] Blazer (if needed)

---

# 5) Grooming “Representation” Checklist (fast upgrades)

### Hair

* [ ] Haircut schedule: every 3–6 weeks
* [ ] 1 styling product
* [ ] Eyebrow tidy (minimal, don’t overpluck)

### Facial hair

* [ ] Either clean shave OR consistent beard length
* [ ] Neckline clean
* [ ] Beard oil (optional)

### Nails & hands

* [ ] Nails trimmed weekly
* [ ] Hand cream if dry

### Teeth & breath

* [ ] Brush 2×
* [ ] Floss 3×/week (upgrade to daily later)
* [ ] Gum/mints for daytime

### Scent

* [ ] Deodorant always
* [ ] One simple fragrance (optional): 1–2 sprays max
* [ ] Clothes actually clean (most important)

* [ ] Walk 10 min daily
* [ ] 2 workouts/week (start) → 3 later
* [ ] Posture cue 3×/day: “chest open, chin back, shoulders down”
* [ ] Mobility 5 min after shower (optional)

## “Not trying too hard” rules (keep you tasteful)

* Avoid loud logos
* Fit and cleanliness > trendy pieces
* Neutral base always works
* One nice layer (jacket/blazer) elevates instantly
* Keep accessories minimal
